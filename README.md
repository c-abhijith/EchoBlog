# Blog API

A FastAPI-based RESTful API for a blogging platform with authentication and authorization.

## Features

- User Authentication (JWT with Access & Refresh Tokens)
- Role-based Authorization (Admin/User)
- User Management
- Blog Posts
- Comments
- Likes
- User Following System
- Protected Routes

## Tech Stack

- FastAPI
- PostgreSQL
- SQLAlchemy
- Pydantic
- JWT Authentication
- Python 3.10+
